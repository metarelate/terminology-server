  • comment "When a max number of productions is specified, this property specify what should happen if the max number is reached. E.g.: if a character is 1-100, we are in an expandDashAdnSuppress pragma, hardLimit is set to 20. If hardLimit cut is not specified, only the entities 1,...,20 are going to be produced. Otherwise none will be produced. " ()
